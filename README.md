# JITU
JITU校园电商平台创立并运营于UKM校区，为校内教师、学生提供一站式商品下单-送货-售后服务。

- 对于每一位入驻商家提供独立的店铺网页，用户无需注册即可下单的购物

- 商家网页后台可对商品信息进行编辑，调整店铺营业时间、配送时间段以及简单的数据可视化等

- 送货员后台可以清晰的查看动态总任务时间表，各宿舍区分货表，进行精准快速的物流配送
